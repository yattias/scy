Μερικές οδηγίες…

Επειδή η φύση των πειραμάτων που πρόκειται να εκτελέσετε, ποικίλει από ομάδα σε ομάδα και από ερώτηση σε ερώτηση, δεν είναι δυνατό να τεθούν συγκεκριμένοι κανόνες για τον σχεδιασμό των πειραμάτων σας. Εντούτοις, λάβετε υπόψη την ακόλουθη λίστα ελέγχου για να αξιολογήσετε την πειραματική σας διαδικασία:

Η πειραματική διαδικασία είναι ακριβής, σαφής και κατανοητή

  • Η λίστα των δραστηριοτήτων είναι ολοκληρωμένη. Όλες οι παράμετροι και τα υλικά έχουν διασαφηνιστεί.
  • Η λίστα των δραστηριοτήτων είναι δομημένη με τέτοιο τρόπο έτσι ώστε να είναι εύκολα κατανοητή. Οι δραστηριότητες θα μπορούσαν να είναι οργανωμένες σε ρουτίνες και υπό ρουτίνες.

Η πειραματική διαδικασία είναι πραγματοποιήσιμη

  • Τα υλικά που έχουν επιλεχτεί είναι προσιτά.
  • Το πείραμα μπορεί να εκτελεσθεί στο διαθέσιμο χρόνο.
  • Οι πιθανοί κίνδυνοι έχουν αξιολογηθεί.

Το πείραμα είναι σύμφωνο με τις ερευνητικές ερωτήσεις και τις υποθέσεις σας.

  • Οι μεταβλητές που περιγράφονται στην ερευνητική/ές ερώτηση/εις αντιστοιχούν με τις παραμέτρους που θα αλλάξουν κατά τη διάρκεια του πειράματος (σχετικότητα).

Το πείραμα είναι σύμφωνο με την επιστημονική μέθοδο.

  • Μόνο μια μεταβλητή θα αλλάζει κάθε φορά.
  • Τα λάθη των μετρήσεων να ελαχιστοποιούνται και οι πηγές των διασπορών να ελέγχονται. Η διασπορά των διαφορετικών τιμών θα πρέπει να είναι όσο το δυνατόν μικρότερη.

Παράδειγμα μιας δραστηριότητας (τα υλικά και οι παράμετροι υπογραμμίζονται): Μετρήστε 50 ml ενός διαλύματος S με ένα σιφώνιο 50 ml.
