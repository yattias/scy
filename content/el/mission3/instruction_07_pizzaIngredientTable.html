07: Πίνακας Συστατικών της Πίτσας

Σημειώστε ότι η δραστηριότητα αυτή είναι προεραιτική. Ρωτήστε τον/την εκπαιδευτικό αν θα πρέπει να κάνετε αυτή την εργασία.

Στην περίπτωση που υπάρχει διαθέσιμος χρόνος και κατάλληλος εξοπλισμός (κινητές συσκευές αποθήκευσης δεδομένων/κινητά τηλέφωνα), μπορείτε να συλλέξετε τις πληροφορίες θρεπτικής αξίας των συστατικών της πίτσας και να συμπληρώσετε τον πίνακα. Μπορείτε να επισκεφτείτε τοπικές υπεραγορές ή φρουταρίες με κινητές συσκευές αποθήκευσης δεδομένων ή να ψάξετε στο διαδίκτυο για να βρείτε τις πληροφορίες. Σε περίπτωση που δεν είναι διαθέσιμες κινητές συσκευές αποθήκευσης δεδομένων, μπορείτε να πάρετε σημειώσεις στο χαρτί. Αφού τα συστατικά στον πίνακα είναι τόσα πολλά, θα είναι πολύ χρονοβόρο να εντοπίσετε πληροφορίες για όλα. Υπάρχουν δύο εναλλακτικές λύσεις σε αυτό το πρόβλημα. Μπορείτε να χωριστείτε σε ομάδες και να ψάξετε πληροφορίες για διαφορετικές ομάδες συστατικών (πχ. 5-10 συστατικά ανά ομάδα)  ή ο/η εκπαιδευτικός μπορεί να σας δώσει έναν πίνακα ο οποίος να έχει αρκετές πληροφορίες που είναι συμπληρωμένες ήδη και να συμπληρώσετε τις πληροφορίες που λείπουν. Θα χρησιμοποιήσετε τον πίνακα αυτό αργότερα, στη Δραστηριότητα 7 για την στρατηγική της βελτιστοποίησης.

Στη συνέχεια: Αφού ολοκληρώσετε τη δραστηριότητα αυτή, επιστρέψτε στο χάρτη της αποστολής και προχωρήστε στο περιβάλλον "Δομή της Τροφικής Πυραμίδας", στη δραστηριότητα "Ερωτήση για την Τροφική Πυραμίδα".
