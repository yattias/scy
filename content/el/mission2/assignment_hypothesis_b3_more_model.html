Διατύπωση υποθέσεων (μοντέλο SCYDynamics)

Στο προηγούμενο στάδιο διατυπώσατε μερικά διερευνήσιμα ερωτήματα σχετικά με την προσομοίωση. Μπορεί να είστε ήδη σε θέση να μαντέψετε το αποτέλεσμα του παιχνιδιού, δηλαδή μπορεί να είστε σε θέση να γράψετε τις προσδοκίες σας για το αποτέλεσμα του παιχνιδιού μοντελοποίησης. Αυτές οι ιδέες θα αναπαριστούν μια πιθανή απάντηση στα διερευνήσιμά σας ερωτήματα. Στα πλαίσια της επιστημονικής έρευνας, αυτές οι προσδοκίες ορίζονται ως «υποθέσεις». Όπως και στην περίπτωση των διερευνήσιμων ερωτημάτων, έτσι και με τις υποθέσεις γίνεται μια προσπάθεια να σχετίζονται με μετρήσιμες μεταβλητές. Αυτό σημαίνει ότι οι υποθέσεις πρέπει να περιλαμβάνουν τις μεταβλητές οι οποίες θα μετρηθούν μέσα από το παιχνίδι της μοντελοποίησης. Ωστόσο, η διατύπωση των υποθέσεων ακολουθεί το πρότυπο μιας πρότασης «αν…., τότε….» (π.χ. «Αν η μεταβλητή 1 αυξηθεί, τότε η μεταβλητή 2 θα μειωθεί»). Αυτό το πρότυπο διατύπωσης υποθέσεων προτιμάται γιατί οι υποθέσεις που διατυπώνονται με προτάσεις του τύπου «αν…., τότε….» μπορούν είτε να γίνουν αποδεκτές είτε να απορριφτούν, γεγονός που προσθέτει στην ισχύ και την αξιοπιστία της πειραματικής διαδικασίας. 

Σημειώστε ότι η κατάλληλη οργάνωση της πειραματικής διαδικασίας δεν προϋποθέτει την επιβεβαίωση των υποθέσεων, αφού αυτό θα εξεταστεί μετά την καταγραφή των δεδομένων που θα προκύψουν από το παιχνίδι προσομοίωσης. Πράγματι, πολλά επιστημονικά πειράματα οδήγησαν σε αξιόλογη γνώση ακριβώς επειδή απόρριψαν μια ερευνητική υπόθεση (π.χ. απόρριψη μιας αναμενόμενης παρενέργειας στην ιατρική). Αυτό που πρέπει να κάνετε εσείς είναι να διατυπώσετε μια υπόθεση για κάθε διερευνήσιμο ερώτημα.

Στο προηγούμενο στάδιο έχετε διατυπώσει μερικά διερευνήσιμα ερωτήματα. Ίσως έχετε ήδη κάποιες ιδέες ή προσδοκίες για τις απαντήσεις στα ερωτήματα σας. Αυτές οι «προσδοκίες» είναι οι υποθέσεις σας. Ο στόχος σε αυτό το στάδιο είναι να διατυπώσετε μια υπόθεση για κάθε διερευνήσιμο ερώτημα. Αν σας ενδιαφέρει να μάθετε πώς γράφετε μια «καλή» υπόθεση τότε διαβάστε μερικές οδηγίες.

Τώρα πρέπει να διατυπώσετε τις υποθέσεις μας.
