Οδηγίες για τη διατύπωση υποθέσεων

1.

Πρώτα πρέπει να προσδιορίσετε τις μεταβλητές που θα συμπεριλαμβάνονται στις υποθέσεις.
2. Μετά πρέπει να διακρίνετε τις μεταβλητές σε ανεξάρτητες και εξαρτημένες. Οι εξαρτημένες μεταβλητές αντιστοιχούν στα αποτελέσματα που παρατηρούνται στα πειράματα, ενώ οι ανεξάρτητες μεταβάλλονται για να παρατηρηθεί η επίδραση τους στις εξαρτημένες μεταβλητές.
3.

Μετά που θα διακρίνετε τις μεταβλητές σε εξαρτημένες και ανεξάρτητες, θα πρέπει να καθορίσετε τις σχέσεις μεταξύ τους, δηλαδή θα πρέπει να σημειώσετε ποιες ανεξάρτητες μεταβλητές θα επηρεάσουν τις εξαρτημένες.

4.

Έπειτα μπορείτε να διατυπώσετε τις υποθέσεις σας με τη μορφή «αν…..τότε….», οι οποίες πρέπει να συνδέουν μια εξαρτημένη με τουλάχιστον μια ανεξάρτητη μεταβλητή. Ένα παράδειγμα μιας είναι υπόθεσης είναι: «Αν η μεταβλητή Β μειωθεί, η μεταβλητή Α θα αυξηθεί».
5.

Χρησιμοποιήστε μόνο μια εξαρτημένη μεταβλητή σε κάθε υπόθεση που διατυπώνετε.

6.

Αν υπάρχουν περισσότερες από μια ανεξάρτητες μεταβλητές που επηρεάζουν την εξαρτημένη, μπορείτε να τις συμπεριλάβετε όλες σε μια πειραματική διαδικασία. Ωστόσο, πρέπει να μεταβάλλετε μόνο μια κάθε φορά και να διατηρείτε τις άλλες σταθερές, έτσι ώστε να είστε σίγουροι ότι η επίδραση που παρατηρείται στην εξαρτημένη μεταβλητή οφείλεται μόνο στη μεταβλητή που αλλάξατε. Σε αυτή την περίπτωση, η διατύπωση της υπόθεσής σας θα πρέπει να είναι όπως το παράδειγμα: «Αν η ανεξάρτητη μεταβλητή Β μειωθεί και όλες οι άλλες διατηρηθούν σταθερές, τότε η εξαρτημένη μεταβλητή Β θα αυξηθεί».
7.

Σημειώστε ότι η κατάλληλη οργάνωση της πειραματικής διαδικασίας δεν προϋποθέτει ότι οι υποθέσεις σας πρέπει να επιβεβαιώνονται. Πολλά επιστημονικά πειράματα οδήγησαν σε πολύτιμες γνώσεις ακριβώς γιατί οι υποθέσεις τους απορρίφθηκαν (π.χ. απόρριψη μιας αναμενόμενης παρενέργειας στην ιατρική).

8.

Μπορεί να παρατηρηθούν στα αποτελέσματα κάποιες παρατηρήσεις οι οποίες δεν έχουν αναφερθεί στις υποθέσεις που έχετε διατυπώσει. Αν θέλετε να ενσωματώσετε αυτές τις απρόβλεπτες παρατηρήσεις στην πειραματική σας διαδικασία, θα πρέπει να ακολουθήσετε τις οδηγίες για τη διατύπωση υποθέσεων από την αρχή. Δηλαδή, θα πρέπει να προσδιορίσετε τις μεταβλητές (1), να τις διακρίνετε σε εξαρτημένες και ανεξάρτητες (2), να διευκρινίσετε τις σχέσεις τους (3), να διατυπώσετε την υπόθεση ως δήλωση της μορφής «αν…τότε…» (4) και να προχωρήσετε σε νέο πειραματισμό για να ελέγξετε τη νέα σας υπόθεση (5). 

9.

Ακόμα και αν έχετε διατυπώσει μια υπόθεση για την οποία ήσαστε βέβαιοι ότι θα επιβεβαιωνόταν, υπάρχει πάντα μια πιθανότητα τα αποτελέσματα να μην είναι τα αναμενόμενα. Αυτός ακριβώς είναι ο τρόπος που η επιστήμη προχωρά και εξελίσσεται. Ιδέες που ίσχυαν και ήταν αξιόπιστες έχουν απορριφθεί ή αντικατασταθεί από καλύτερες ιδέες.
