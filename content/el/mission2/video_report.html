Βίντεο- αναφορά

Κατά τη διάρκεια αυτής της αποστολής θα πραγματοποιήσετε ένα ή περισσότερα πειράματα που έχουν να κάνουν με τα υδάτινα οικοσυστήματα. Αντί να γράψετε μια παραδοσιακή, γραπτή αναφορά στο χαρτί για αυτά τα πειράματα, θα δημιουργήσετε ένα βίντεο-αναφορά. Σε αυτό το βίντεο-αναφορά θα επικεντρωθείτε σε ένα συγκεκριμένο πείραμα που θα έχετε πραγματοποιήσει. Θα εξηγείτε στους άλλους (δηλαδή στην υπόλοιπη τάξη, στους φίλους και στην οικογένεια) τι πείραμα έχετε κάνει και ποια ήταν τα αποτελέσματα.

  • Δημιουργήστε ομάδες των τεσσάρων.
  • Αποφασίστε σε ποιο πείραμα, και από ποιο θέμα, θα επικεντρωθείτε στο βίντεο σας.
  • Το βίντεο σας πρέπει να αποτελείται από τα πιο κάτω:
    1. Εισαγωγή του προβλήματος
    2. Εξηγήστε τη συγκεκριμένη κατάσταση  ή πρόβλημα που αποτέλεσε το αρχικό σημείο της διερεύνησης σας. Παρουσιάστε μερικές γενικές πληροφορίες όπου είναι απαραίτητο.
    3. Διερευνήσιμα ερωτήματα και υποθέσεις
      4. Διατυπώστε ξεκάθαρα τα διερευνήσιμα ερωτήματα και τις υποθέσεις σας.
    4. Πειραματική διαδικασία
      5. Περιγράψτε το πείραμα που έχετε πραγματοποιήσει. Μπορείτε να βιντεογραφήσετε την πειραματική διάταξη στην τάξη ή στο εργαστηρίο, ή να δείξετε το μοντέλο στον υπολογιστή, με το οποίο εργαστήκατε.
    5. Αποτελέσματα του πειράματος
      6. Συζητήστε εν συντομία τα αποτελέσματα του πειράματός σας. Μπορείτε να ενσωματώσετε εικόνες από τις γραφικές παραστάσεις και τα διαγράμματα που έχετε δημιουργήσει.
    6. Συμπεράσματα
      7. Συζητήστε εν συντομία τα κύρια συμπεράσματα από τις διερευνήσεις σας. Κάντε επίσης μερικές αναφορές στο γενικό πρόβλημα που έχει αναφερθεί στην εισαγωγή.
  • Η συνολική διάρκεια του βίντεο σας δεν πρέπει να ξεπερνά τα 5 λεπτά.
  • Όταν γράφετε μια αναφορά, πιθανό να ξοδεύετε κάποιο χρόνο για τη διατύπωση των προτάσεων σας και για να σκεφτείτε ποιες συγκεκριμένες πληροφορίες πρέπει να συμπεριλάβετε στην αναφορά και ποιες όχι. Θα κάνετε το ίδιο και εδώ. Έτσι αντί να ξεκινήσετε τη δημιουργία του βίντεο σας out of the blue, θα γράψετε μια πολύ απλή περιγραφή για κάθε σκηνή που θα συμπεριλαμβάνεται στο βίντεο σας. Η περιγραφή σας αυτή θα πρέπει να περιλαμβάνει μερικές σημαντικές λέξεις κλειδιά που μπορείτε να χρησιμοποιήσετε σαν συνδέσεις καθώς προβάλλεται το βίντεο. Είναι καλή ιδέα να συζητήσετε για την περιγραφή αυτή με τον εκπαιδευτικό.
  • Μια τελευταία σημείωση για αξιολόγηση: ο κύριος στόχος του βίντεο-αναφορά είναι να πληροφορήσετε το κοινό σας για τη δουλειά που έχετε κάνει. Είναι φυσικά επιτρεπτό να αξιοποιήσετε τη δημιουργικότητα και την αίσθηση του χιούμορ, αλλά όχι εις βάρος του βασικού στόχο του βίντεο που είναι η πληροφόρηση.
