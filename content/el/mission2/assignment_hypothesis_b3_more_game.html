Διατύπωση υποθέσεων (παιχνίδι μοντελοποίησης)

Στο προηγούμενο στάδιο διατυπώσατε μερικά διερευνήσιμα ερωτήματα σχετικά με το παιχνίδι θηρευτή-θηράματος. Μπορεί να είστε ήδη σε θέση να μαντέψετε το αποτέλεσμα του παιχνιδιού, δηλαδή μπορεί να είστε σε θέση να προβλέψετε πώς θα εξελιχθούν οι πληθυσμοί του θηρευτή και του θηράματος.. Αυτές οι ιδέες θα αναπαριστούν μια πιθανή απάντηση στα διερευνήσιμα σας ερωτήματα. Στα πλαίσια της επιστημονικής έρευνας, αυτές οι προσδοκίες ορίζονται ως «υποθέσεις». Όπως και στην περίπτωση των διερευνήσιμων ερωτημάτων, έτσι και με τις υποθέσεις γίνεται μια προσπάθεια να σχετίζονται με μετρήσιμες μεταβλητές. Αυτό σημαίνει ότι οι υποθέσεις πρέπει να περιλαμβάνουν τις μεταβλητές οι οποίες θα μετρηθούν μέσα από το παιχνίδι θηρευτή-θηράματος. Ωστόσο, η διατύπωση των υποθέσεων ακολουθεί το πρότυπο μιας πρότασης «αν…., τότε….» (π.χ. «Αν η μεταβλητή 1 αυξηθεί, τότε η μεταβλητή 2 θα μειωθεί»). Αυτό το πρότυπο διατύπωσης υποθέσεων προτιμάται γιατί οι υποθέσεις που διατυπώνονται με προτάσεις του τύπου «αν…., τότε….» μπορούν είτε να γίνουν αποδεκτές είτε να απορριφθούν, γεγονός που προσθέτει στην ισχύ και την αξιοπιστία της πειραματικής διαδικασίας.

Σημειώστε ότι η ορθή οργάνωση της πειραματικής διαδικασίας δεν προϋποθέτει την επιβεβαίωση των υποθέσεων, αφού αυτό θα εξεταστεί μετά την καταγραφή των δεδομένων που θα προκύψουν από την προσομοίωση SCYDynamics. Πράγματι, πολλά επιστημονικά πειράματα οδήγησαν σε σημαντικά αποτελέσματα ακριβώς επειδή απέρριψαν μια ερευνητική υπόθεση (π.χ. απόρριψη μιας αναμενόμενης παρενέργειας στην ιατρική). Αυτό που πρέπει να κάνετε εσείς είναι να διατυπώσετε μια υπόθεση για κάθε διερευνήσιμο ερώτημα.

Στο προηγούμενο βήμα, έχετε διατυπώσει μερικά διερευνήσιμα ερωτήματα. Ίσως να έχετε ήδη κάποιες ιδέες ή προσδοκίες για τις απαντήσεις στα ερωτήματά σας. Αυτές οι «προσδοκίες» είναι οι υποθέσεις σας. Ο στόχος σε αυτό το στάδιο είναι να διατυπώσετε μια υπόθεση για κάθε διερευνήσιμο ερώτημα.

Αν σας ενδιαφέρει πώς γράφεται μια αποδεκτή υπόθεση, τότε μπορείτε να διαβάσετε τις παρακάτω οδηγίες.

Τώρα πρέπει να διατυπώσετε τις υποθέσεις σας.
