Guidelines: Research questions

1. First, you should identify the variables which will be included in the research questions.
2. Then, you need to distinguish between dependent and independent variables. The former correspond to the observed result of experiments, while the latter are parameters you manipulate to observe their impact on dependent variables.
3. Use only one dependent variable in each research question you formulate.
4. If there are more than one independent variables which influence the dependent variable, you can include them all in the experimental procedure. However, you have to change only one independent variable at a time and keep all other independent variables unchanged, so that you can be sure that the influence you measure on the dependent variable can be attributed to the independent variable you have manipulated.
5. Choose a question that is not too broad and not too narrow.
6. A question has to be researchable (e.g. you have to be able to collect the right data) and feasible using available resources (time, equipment, etc.).
7. 'Open' questions are usually more useful than 'closed' questions.
8. A useful phrase could be: "what is the relation between [concept 1] and [concept 2]?"
