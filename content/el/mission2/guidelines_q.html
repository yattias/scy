Οδηγίες: Ερευνητικά ερωτήματα

1. Πρώτα πρέπει να προσδιορίσετε τις μεταβλητές που θα συμπεριλαμβάνονται στις υποθέσεις.
2.

Μετά πρέπει να διακρίνετε τις μεταβλητές σε ανεξάρτητες και εξαρτημένες. Οι εξαρτημένες μεταβλητές αντιστοιχούν στα αποτελέσματα που παρατηρούνται στα πειράματα, ενώ οι ανεξάρτητες μεταβάλλονται για να παρατηρηθεί η επίδραση τους στις εξαρτημένες μεταβλητές.
3.

Χρησιμοποιήστε μόνο μια εξαρτημένη μεταβλητή σε κάθε διερευνήσιμο ερώτημα που διατυπώνετε.
4.

Αν υπάρχουν περισσότερες από μια ανεξάρτητες μεταβλητές που επηρεάζουν την εξαρτημένη, μπορείτε να τις συμπεριλάβετε όλες στην πειραματική διαδικασία. Ωστόσο, πρέπει να μεταβάλλετε μόνο μια κάθε φορά και να διατηρείτε τις άλλες σταθερές, έτσι ώστε να είστε σίγουροι ότι η επίδραση που παρατηρείτε στην εξαρτημένη μεταβλητή οφείλεται μόνο στην μεταβλητή που αλλάξατε.
5.

Διαλέξτε ένα ερώτημα που δεν είναι ούτε υπερβολικά ανοικτού τύπου αλλά ούτε κλειστού τύπου.

6.

Ένα ερώτημα πρέπει να είναι διερευνήσιμο  (π.χ. θα πρέπει να είστε σε θέση να συλλέξετε τα σωστά δεδομένα) και εφικτό ως προς τη χρήση των μέσων (χρόνος, εξοπλισμός, κ.λπ.).

7.

Οι «ανοικτού τύπου» ερωτήσεις είναι συνήθως πιο χρήσιμες από ό, τι οι «κλειστού τύπου» ερωτήσεις.

8.

Μια χρήσιμη φράση θα μπορούσε να είναι: «Ποια είναι η σχέση μεταξύ της [έννοιας 1] και της [έννοιας 2];»
