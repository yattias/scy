Οδηγίες για το βίντεο-αναφορά

Μπορείτε να χρησιμοποιήσετε οποιαδήποτε μηχανή ή λογισμικό επεξεργασίας για τη διαδικασία δημιουργίας του βίντεο. Όταν το βίντεο ολοκληρωθεί, ανεβάστε το στο YouTube και στη συνέχεια δημιουργήστε ένα σύνδεσμο στο SCYLab.

Κοιτάξτε τις οδηγίες χρήσης της μηχανής σας.

Επεξεργασία του βίντεο

Υπάρχουν πολλά διαφορετικά προγράμματα που μπορείτε να χρησιμοποιήσετε για να επεξεργαστείτε το βίντεο σας. Ένα εύχρηστο πρόγραμμα που μπορείτε να χρησιμοποιήσετε είναι το Windows Movie Maker. Μερικές βασικές οδηγίες για το Windows Movie Maker δίνονται εδώ.

Ανέβασμα του βίντεο στο YouTube

Πηγαίνετε στο www.youtube.com. Για να ανεβάσετε το βίντεο πρέπει να έχετε λογαριασμό στο YouTube. Αν δεν έχετε λογαριασμό, επιλέξτε το κουμπί «Δημιουργία Λογαριασμού» στο πάνω μέρος του παραθύρου. Όταν συνδεθείτε, επιλέξτε το κουμπί «Ανέβασμα Βίντεο». Στην επόμενη σελίδα μπορείτε να εντοπίσετε το βίντεο που θέλετε να ανεβάσετε, να του δώσετε όνομα και να διαμορφώσετε τις προτιμήσεις σας. Σημαντικό: εισάγετε την ετικέτα “SCY ECO Mission”. Με αυτό τον τρόπο, εσείς (και οι άλλοι) θα μπορείτε να βρείτε τα βίντεο σας όταν τα ανεβάσετε.

Δημιουργία συνδέσμου στο SCYLab
