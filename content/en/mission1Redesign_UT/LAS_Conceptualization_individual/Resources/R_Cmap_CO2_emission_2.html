2. Create a link between nodes

Always connect the nodes in your concept map to each other.

Release the mouse when a green border appears around the second node.

Top of page

3. Remove a node or link

A popup window appears. Click the Yes button if you want to remove the node.


Note that the link is removed automatically.

Follow the same steps to remove a link (while keeping the nodes). The only difference is, of course, that you have to click on the to-be-removed link instead of the node.
