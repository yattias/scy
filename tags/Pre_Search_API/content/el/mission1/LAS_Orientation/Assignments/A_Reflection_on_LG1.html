Δραστηριότητα αναστοχασμού

Ο στόχος αυτής της εργασίας είναι να ασχοληθείτε σοβαρά με τις έννοιες και τις δεξιότητες, που θα πρέπει να εξετάσετε κατά τη διάρκεια αυτής της αποστολής. Βαθμολογήστε με βάση αυτόν τον πίνακα αξιολόγησης για την δική σας περίπτωση μέχρι αυτή τη στιγμή και θα αξιολογήσετε ακόμη μια φορά στο τέλος της αποστολής με βάση αυτόν τον πίνακα, έτσι ώστε να έχετε μια ιδέα για την πρόοδο σας. Να είστε ειλικρινείς. Αυτή η δραστηριότητα δεν επηρεάζει τον/τους βαθμό/βαθμούς που θα πάρετε σε αυτή την αποστολή με κανένα τρόπο, αφού την αντιμετωπίζετε με σοβαρότητα.

Δραστηριότητα:

  1. Στην αριστερή στήλη θα βρείτε μερικές δεξιότητες και έννοιες. Διαβάστε τις περιγραφές στις τρεις στήλες που βρίσκονται στα δεξιά (ΕΠΙΠΕΔΟ 1, ΕΠΙΠΕΔΟ 2 και ΕΠΙΠΕΔΟ 3).
  2. Προσδιορίστε ποια περιγραφή είναι η εγκυρότερη για την δική σας περίπτωση μέχρι στιγμής, σημειώνοντας σε μια θέση στην κλίμακα, με ένα σταυρό. Π.χ.  ένα αποτέλεσμα 30% θα σήμαινε ότι είστε κάπου ανάμεσα στο επίπεδο 1 και 2. Και ένα αποτέλεσμα 85% θα σήμαινε ότι η περιγραφή κάτω από το επίπεδο 3 είναι αρκετά ακριβής για εσάς.
