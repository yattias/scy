Καλωσορίσατε στο SCY-Lab!

Στην κεντρική οθόνη μπορείτε να δείτε το κεντρικό Αναδυόμενο Μαθησιακό Προϊόν (τελικό Α.ΜΑ.Π.) στο οποίο θα εργαστείτε αυτή τη στιγμή. Εδώ, θα διαβάσετε μια περιγραφή της πρόκλησης που θα αντιμετωπίσετε σε αυτή την αποστολή.

Στα αριστερά μπορείτε να δείτε άλλα Αναδυόμενα Μαθησιακά Προϊόντα, τα οποία θα σας βοηθήσουν να ολοκληρώσετε την αποστολή σας. Εδώ, μπορείτε να βρείτε ένα βίντεο με την πρόκληση. Εκεί θα βρείτε επίσης ένα γλωσσάρι το οποίο παραθέτει τις κεντρικές έννοιες που θα χρειαστεί να κατανοήσετε προκειμένου να ολοκληρώσετε την αποστολή σας.

Καθώς προχωράτε πιο βαθιά στο “Scy-lab”, θα εργαστείτε με πολλαπλά Α.ΜΑ.Π. τα οποία θα σας ανατίθενται πάνω από την κεντρική οθόνη. Προς το παρόν δεν υπάρχουν αντικείμενα να βρείτε εκεί ακόμα.

Στις τέσσερις γωνιές της πλατφόρμας του “Scy-lab”, μπορείτε να βρείτε συγκεκριμένες υπηρεσίες του Scy που θα χρειαστείτε κατά τη διάρκεια της αποστολής.
  • Στην πάνω αριστερή γωνιά της πλατφόρμας μπορείτε να επιλέξετε συμμαθητές σας για να συνεργαστείτε μαζί τους σε ένα συγκεκριμένο Α.ΜΑ.Π. Για να προσκαλέσετε ένα συμμαθητή σας για να συνεργαστείτε σε ένα συγκεκριμένο Α.ΜΑ.Π., απλά μεταφέρετε τον συμμαθητή σας στο αντίστοιχο Α.ΜΑ.Π.
  • Στην κάτω αριστερή γωνιά της πλατφόρμας μπορείτε να δημιουργήσετε νέα Α.ΜΑ.Π. καθώς και να ψάξετε για Α.ΜΑ.Π. που ήδη έχετε.
  • Στην πάνω δεξιά γωνιά της πλατφόρμας μπορείτε να ενεργοποιήσετε ένα εργαλείο προγραμματισμού δραστηριοτήτων (planning tool) για να προγραμματίσετε την εργασία σας στα Α.ΜΑ.Π.
  • Στην κάτω δεξιά γωνιά της πλατφόρμας μπορείτε να δείτε τον χάρτη της αποστολής. Όλα τα Τελικά Αναδυόμενα Μαθησιακά Προϊόντα (Τ.Α.ΜΑ.Π.) οπτικοποιούνται εκεί και πατώντας σε καθένα από αυτά εμφανίζεται το αντίστοιχο Τ.Α.ΜΑ.Π. στο κέντρο της οθόνης. Με αυτόν τον τρόπο μπορείτε να πλοηγηθείτε στην πλατφόρμα του “Scy-lab”.

Στην δεξιά πλευρά της οθόνης σας μπορείτε να δείτε περιγραφές της παρούσας εργασίας σας όπως αυτή εδώ. Αυτή τη στιγμή η εργασίας σας είναι να εξοικειωθείτε με το περιβάλλον του  “Scy-lab” και με την αποστολή που θα αντιμετωπίσετε. Για να κάνετε το δεύτερο, διαβάστε το κείμενο με την εργασία που βρίσκεται στο κέντρο της οθόνης σας και ανοίξτε το βίντεο που βρίσκεται στο αριστερό τμήμα της οθόνης σας.
