  • Γιατί ένα σπίτι φιλικό προς το περιβάλλον ως προς τις εκπομπές του διοξειδίου του άνθρακα;
  • Ποια είναι η πρόκληση;

Ποια είναι η πρόκληση;

Η αποστολή σας είναι να σχεδιάσετε ένα σπίτι φιλικό προς το περιβάλλον ως προς τις εκπομπές του διοξειδίου του άνθρακα. Συνεπώς θα πρέπει να σκεφτείτε για όλα τα είδη μέτρων και εφαρμογών, τα οποία θα μειώσουν τις εκπομπές του διοξειδίου του άνθρακα από το σπίτι σας. Ωστόσο, παρόλο που η μείωση των εκπομπών του διοξειδίου του άνθρακα είναι το κεντρικό θέμα, το σπίτι σας θα πρέπει να παρέχει χώρο και ένα λογικό επίπεδο άνεσης για μια οικογένεια τεσσάρων ατόμων (δύο γονείς και δύο παιδιά).

Στο τέλος αυτής της αποστολής πρέπει να παρουσιάσετε στην υπόλοιπη τάξη το δικό σας σπίτι που είναι φιλικό στο περιβάλλον ως προς τις εκπομπές διοξειδίου του άνθρακα. Κατά τη διάρκεια της αποστολής θα πρέπει να ολοκληρώσετε ένα αριθμό εργασιών. Ο Χάρτης Αποστολής (στην δεξιά γωνιά της οθόνης σας) θα σας καθοδηγήσει στην ολοκλήρωση της αποστολής. Θα εργαστείτε για το σχεδιασμό του δικού σας σπιτιού σε μικρές ομάδες (ομάδα σχεδιασμού).

Προτού αρχίσετε να σχεδιάζετε το σπίτι σας, θα πρέπει να γίνετε ειδικός σε ένα τομέα. Σε κάθε μέλος της ομάδας σχεδιασμού θα ανατεθεί ένας συγκεκριμένος ρόλος ειδικού. Θα είστε είτε (1) ειδικός για τη θερμομόνωση είτε (2) ειδικός για την ενέργεια είτε (3) ειδικός για οικιακά θέματα. Ειδικά στο πρώτο μισό της αποστολής, θα συνεργαστείτε με συμμαθητές σας που θα έχουν τον ίδιο ρόλο ειδικού. Το βασικό καθήκον της ομάδας ειδικών είναι να βρει πληροφορίες, καθώς και να κάνει πραγματικά και εικονικά πειράματα. Τα αποτελέσματά σας θα τα παρουσιάσετε σε έναν εννοιολογικό χάρτη.
