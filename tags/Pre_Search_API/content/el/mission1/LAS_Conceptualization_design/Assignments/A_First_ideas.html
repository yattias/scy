Αρχικές ιδέες της ομάδας σχεδιασμού σας

Τι κάνει ένα σπίτι φιλικό προς το περιβάλλον ως προς τις εκπομπές του διοξειδίου του άνθρακα;

Ο σκοπός σας είναι να σχεδιάσετε ένα σπίτι φιλικό προς το περιβάλλον. Όμως τι μέτρα μπορεί να πάρει κάποιος προκειμένου να φτιάξει ένα σπίτι φιλικό προς το περιβάλλον; Χρησιμοποιήστε το σχέδιο του σπιτιού για να σημειώσετε τις πρώτες ιδέες της ομάδας σχεδιασμού σας (design group). Σε αυτή τη δραστηριότητα δεν υπάρχουν σωστές και λάθος απαντήσεις. Η ιδέα είναι απλά να γίνει μια πρώτη ιδεοθύελλα για το σπίτι που είναι φιλικό προς το περιβάλλον.  Συνεπώς οι ιδέες σας μπορούν να είναι όσο αυθόρμητες και δημιουργικές θέλετε.

Όλες οι ιδέες σας να τοποθετηθούν μέσα και γύρω από το σχέδιο του σπιτιού που βλέπετε εδώ. Στο σχέδιο αυτό υπάρχουν τρία βέλη. Αυτά τα βέλη θα σας βοηθήσουν να οργανώσετε τις ιδέες σας. Για παράδειγμα, μια ιδέα σας μπορεί να σχετίζεται με το είδος της ενέργειας που ρέει μέσα στο σπίτι (αριστερό βέλος), το μέγεθος της ενέργειας που καταναλώνεται μέσα στο σπίτι (ενδιάμεσο βέλος), ή το ποσό της θερμότητας που χάνεται (δεξί βέλος). Συνεπώς θα πρέπει να αποφασίσετε με ποιον τρόπο η κάθε ιδέα σας συνεισφέρει στο να κάνει το σπίτι σας πιο φιλικό προς το περιβάλλον και να την τοποθετήσετε πάνω από το αντίστοιχο βέλος στο σχέδιο.
