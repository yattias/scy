Κριτική εξέταση και σύγκριση των πηγών

Βρίσκοντας διάφορες πηγές με τις κατάλληλες πληροφορίες, πρέπει να συλλέξετε και να οργανώσετε τα συμπεράσματά σας με ένα ικανοποιητικό τρόπο. Για βοήθεια θα ήταν καλό να ακολουθήσετε τις πιο κάτω οδηγίες:

Διαβάστε

Πρώτα πρέπει να διαβάσετε προσεκτικά τις πηγές που έχετε βρει. Μπορείτε βέβαια να προσπεράσετε κάποια τμήματα που δεν έχουν καμία σχέση με κανένα από τα ερευνητικά σας ερωτήματα.

Συνοψίστε

Κατόπιν πρέπει να συνοψίσετε τις πληροφορίες που χρειάζεστε. Αντί να γράψετε μια περίληψη σε συνεχές κείμενο, θα μπορούσατε επίσης να προσπαθήσετε να συντάξετε έναν κατάλογο λέξεων κλειδιών ή βασικών προτάσεων. ΜΗΝ αντιγράφεται ολόκληρες παραγράφους (αντίγραφα) από τα υπάρχοντα κείμενα.

Οργανώστε τα δεδομένα

Μερικές φορές είναι πιο σωστό να οργανωθούν οι συγκεκριμένες πληροφορίες σε πίνακες (π.χ. κατάλογος συσκευών και οι ιδιότητες τους, κ.λπ.).
