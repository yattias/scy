Ατομική αναφορά

Γράψτε μια αναφορά για τον δήμαρχο της πόλης σας

Έχετε μάθει πολλά πράματα κατά τη διάρκεια αυτής της αποστολής. Έχετε γίνει ειδικοί, έχετε ανταλλάξει τη γνώση που αποκτήσατε ως ειδικοί με τα υπόλοιπα μέλη της ομάδας σχεδιασμού σας και μαζί σχεδιάσατε το οικολογικό σας σπίτι. Τώρα η εργασίας σας είναι να γράψετε μια ατομική αναφορά για να παρουσιάσετε τη μελέτη Βιοενέργειας (BioEnergy’s study) στον δήμαρχο της πόλης σας. Η αναφορά σας θα πρέπει να είναι και ενημερωτική και εύκολη να διαβαστεί, έτσι ώστε ο δήμαρχος (και πιθανοί κάτοικοι) της πόλης σας να είναι εύκολο να το καταλάβει/ουν.
