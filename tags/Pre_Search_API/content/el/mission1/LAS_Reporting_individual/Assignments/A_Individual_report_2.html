Η αναφορά σας θα αποτελείται από (τουλάχιστον) τα ακόλουθα μέρη:


Εισαγωγή

Γιατί να μειωθούν τα παγκόσμια επίπεδα διοξειδίου του άνθρακα;
Εξηγήστε τις αιτίες και τους πιθανούς κινδύνους των αυξανόμενων επιπέδων διοξειδίου του άνθρακα. Μπορείτε να χρησιμοποιήσετε τον εννοιολογικό σας χάρτη προκειμένου να υποστηρίξετε το κείμενό σας.

Ο σχεδιασμός του σπιτιού σας

Παρουσιάστε τον τελικό (Βιοενεργειακό) σχεδιασμό του σπιτιού σας. Συμπεριλάβετε τις σημαντικότερες εικόνες και σχέδια σπιτιών, παρουσιάστε (και εξηγήστε) τα δεδομένα από την προσομοίωση “Thermal simulation”, την προσομοίωση κατανάλωσης ηλεκτρικής ενέργειας “Electricity consumption simulation” και τον ενεργειακό μετατροπέα “Energy converter”. Επίσης κάντε μια λίστα επιλογών που έγιναν από την ομάδα σχεδιασμού σας και εξηγήστε το συλλογισμό σας.

Λίστα πηγών

Μία λίστα όλων των πηγών (βιβλίων, ιστοσελίδων κ.ά.) που έχετε χρησιμοποιήσει κατά τη διάρκεια της αποστολής.

Παράρτημα

Ως παράρτημα, θα πρέπει να προσθέσεις μια συνοπτική αναφορά σχετικά με το πείραμα που έκανες εσύ ή κάποιο άλλο μέρος της ομάδας σχεδιασμού σας. Περιγράψτε τα ερευνητικά ερωτήματα, τις υποθέσεις, τις πειραματικές διαδικασίες, τα αποτελέσματα και τα συμπεράσματα.
