Πραγματοποιήστε πειράματα

Τώρα είναι η στιγμή να πραγματοποιηθούν τα πειράματά σας. Αν έχετε περάσει από όλη την προπαρασκευαστική διαδικασία με σοβαρότητα, αυτό το στάδιο δεν θα είναι πάρα πολύ δύσκολο.

Γενικές οδηγίες είναι:

  • δουλέψτε με σοβαρότητα και ακρίβεια
  • αποθηκεύσετε όλα τα δεδομένα σας προσεκτικά (αυτά είναι τα πρωτογενή δεδομένα σας)
  • πάρτε σημειώσεις για οποιεσδήποτε αποκλίσεις από τον αρχικό σας πειραματικό σχεδιασμό
