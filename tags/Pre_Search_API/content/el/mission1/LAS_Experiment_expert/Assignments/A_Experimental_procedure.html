Σχεδιασμός πειραμάτων

Προτού να αρχίσετε πραγματικά να κάνετε τα πειράματά σας, θα πρέπει να περιγράψετε τι προγραμματίζετε να κάνετε. Με άλλα λόγια, θα πρέπει να γράψετε την πειραματική διαδικασία. Μια πειραματική διαδικασία είναι μια λεπτομερής, βήμα προς βήμα περιγραφή ενός πειράματος. Ακόμα και αν σκοπεύετε να κάνετε το πείραμα μόνοι σας, η πειραματική σας διαδικασία θα πρέπει να είναι ακριβής και σαφής. Θα πρέπει να δίνονται αρκετές πληροφορίες έτσι ώστε ένα άλλο άτομο να είναι σε θέση να κάνει το πείραμά σας με τον ίδιο ακριβώς τρόπο.

Μια πειραματική διαδικασία αποτελείται από μια οργανωμένη λίστα δραστηριοτήτων που θα πρέπει να διεκπεραιωθούν. Όλες οι οδηγίες θα πρέπει να διευκρινίζουν τα υλικά που θα πρέπει να χρησιμοποιηθούν και όλες τις παραμέτρους που καθορίζουν τη δραστηριότητα που πρέπει να γίνει (π.χ. οι ποσότητες των ουσιών, οι εισερχόμενες παράμετροι μιας συσκευής, κ.λπ.).

Χρησιμοποιήστε το εργαλείο σχεδιασμού πειραματικής διαδικασίας (Experimental Procedure Tool) προκειμένου να σχεδιάσετε το πείραμά σας. Όπως θα δείτε, αρχικά θα πρέπει να εισάγετε τα ερευνητικά ερωτήματα και την υπόθεση που θα εξεταστεί στο πείραμά σας. Κατόπιν, θα πρέπει να διευκρινίσετε όλα τα βήματα του πειράματός σας και τα υλικά που θα χρειαστείτε προκειμένου να το εκτελέσετε.
