Πραγματοποιήστε εικονικά πειράματα

Μπορεί να έχετε προετοιμάσει (και πραγματοποιήσει) μερικά πραγματικά πειράματα, αλλά δεδομένου ότι ανήκετε στην ομάδα ειδικών για τη Θερμότητα, μπορείτε επίσης να πραγματοποιήσετε μερικά εικονικά πειράματα χρησιμοποιώντας την προσομοίωση Thermal simulation. Μπορείτε να διαβάσετε τον οδηγό χρήσης για να καταλάβετε τι μπορείτε να κάνετε με αυτή την προσομοίωση. Στο τέλος του πειράματος σας, αποθηκεύσετε το σύνολο των δεδομένων σας και θα εμφανίζονται στην οθόνη του SCY-Lab. Κατόπιν, θα πρέπει να επεξεργαστείτε τα δεδομένα σας.
