Πραγματοποιήστε εικονικά πειράματα

Μπορεί να έχετε προετοιμάσει (και πραγματοποιήσει) μερικά πραγματικά πειράματα, αλλά δεδομένου ότι ανήκετε στην ομάδα ειδικών για τη οικιακή χρήση ενέργειας, μπορείτε επίσης να πραγματοποιήσετε μερικά εικονικά πειράματα χρησιμοποιώντας την προσομοίωση Energy consumption simulation (http://bio.edu.ee/flash/electricity/), για την κατανάλωση ενέργειας.

Μπορείτε να διαβάσετε τον οδηγό χρήσης για να καταλάβετε τι μπορείτε να κάνετε με αυτή την προσομοίωση (ο οδηγός χρήσης ενσωματώνεται στο εργαλείο). Στο τέλος του πειράματος σας, αποθηκεύσετε το σύνολο των δεδομένων σας και θα εμφανίζονται στην οθόνη του SCY-Lab. Κατόπιν, θα πρέπει να επεξεργαστείτε τα δεδομένα σας.
