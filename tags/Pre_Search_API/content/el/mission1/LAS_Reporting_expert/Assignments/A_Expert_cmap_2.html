• Εννοιολογικός χάρτης των ειδικών
  • Πώς να αρχίσετε;
  • Παράδειγμα εννοιολογικού χάρτη

Πώς να αρχίσετε;

  1. Αρχίστε τοποθετώντας ένα μικρό αριθμό εννοιών που είναι σημαντικές για τους ειδικούς στον εννοιολογικό χάρτη.
  2. Στη συνέχεια, προσπαθήστε να συνδέσετε αυτές τις έννοιες μεταξύ τους. Το βέλος, το οποίο συνδέει τις δυο έννοιες πρέπει επίσης να έχει μια ετικέτα: μια λέξη ή μια σύντομη φράση που καθορίζει τη σχέση μεταξύ αυτών των εννοιών. (π.χ. «απελευθερώνεται από» ή «παράγει»).
  3. Κάθε έννοια ή ιδέα, πρέπει να συνδέεται τουλάχιστον με μια άλλη ιδέα. 
  4. Από αυτό το σημείο και μετά, ο εννοιολογικός σας χάρτης θα μεγαλώσει. Πιθανόν να σκεφτείτε νέες  έννοιες, οι οποίες σχετίζονται με αυτές που είναι ήδη στον εννοιολογικό χάρτη.
