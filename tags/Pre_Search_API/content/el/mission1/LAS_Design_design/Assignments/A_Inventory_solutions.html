Κατάλογος των λύσεων των ειδικών

Τι θα κάνει το σπίτι σας φιλικό προς το περιβάλλον;

Ο στόχος σας είναι να σχεδιάσετε ένα σπίτι φιλικό προς το περιβάλλον. Στα προηγούμενα μαθήματα δουλέψατε με ομάδες ειδικών προκειμένου να κερδίσετε γνώση σχετικά με όλα τα είδη μέτρων που μπορεί να επηρεάζουν τη φιλικότητα ενός σπιτιού προς το περιβάλλον. Καθένας από εσάς έχει δουλέψει σε ένα συγκεκριμένο πεδίο γνώσης: θερμική μόνωση, παραγωγή ενέργειας ή οικιακά ζητήματα. Τώρα ήρθε η ώρα να επιστρέψετε στις αρχικές σας ομάδες και να ξεκινήσετε πραγματικά να σχεδιάζετε (δηλ. σχεδιασμό, περιγραφή, προσομοίωση) του ΔΙΚΟΥ ΣΑΣ ΣΠΙΤΙΟΥ.

Για να σας βοηθήσουμε να ξεκινήσετε, η ομάδα σχεδιασμού σας θα κάνει μια λίστα με όλες τις πιθανές λύσεις και προτάσεις και από τις τρεις ομάδες ειδικών, για ένα σπίτι φιλικό προς το περιβάλλον. Όπως στη δραστηριότητα με την ιδεοθύελλα στην αρχή της αποστολής, θα χρησιμοποιήσετε το σκίτσο του σπιτιού σας προκειμένου να κάνετε αυτόν τον κατάλογο. Διευθετήστε όλες τις ιδέες γύρω (και μέσα) από το σκίτσο. Συνδέστε το κάθε αντικείμενο με ένα από τα βέλη του σκίτσου.
