Επιλογές σπιτιών

Αποφασίστε ποια θα είναι τα χαρακτηριστικά του σπιτιού σας που είναι φιλικό προς το περιβάλλον

Στην αρχή της αποστολής είχατε κάνει μια ιδεοθύελλα με την ομάδα σχεδιασμού σας, προκειμένου να καταγράψετε τις «αρχικές σας ιδέες». Στο τέλος της φάσης των ειδικών κάνατε έναν «κατάλογο λύσεων από τους ειδικούς», συνδυάζοντας τα ευρήματα και των τριών ομάδων ειδικών. Για αυτή τη δραστηριότητα θα χρησιμοποιήσετε το ίδιο εργαλείο, με το σκίτσο του σπιτιού για τελευταία φορά. Αυτή τη φορά μόνο οι λύσεις και οι ιδέες που αποφασίσετε πραγματικά να χρησιμοποιηθούν για το σπίτι σας θα παρουσιαστούν στο σκίτσο.

Χρησιμοποιήστε το σκίτσο του σπιτιού για να παρουσιάσετε τα χαρακτηριστικά του δικού σας σχεδιασμού. Θα ήταν καλή ιδέα να ξεκινήσετε με τον «κατάλογο λύσεων από τους ειδικούς» και να σβήσετε από αυτόν όλες τις λύσεις που η ομάδα σχεδιασμού σας θα αποφασίσει να μην χρησιμοποιήσει. Συνεπώς, κάντε μια συλλογή όλων των συσκευών που θα πρέπει να υπάρχουν στο σπίτι σας. Επίσης, προσπαθήστε να πάρετε μια απόφαση σχετικά με την/τις πηγή/ές του σπιτιού σας.

Φυσικά θα πρέπει επίσης να συζητήσετε σχετικά με το σχήμα, το μέγεθος κι άλλα αρχιτεκτονικά χαρακτηριστικά του σπιτιού σας και να κάνετε κάποιες επιλογές.
