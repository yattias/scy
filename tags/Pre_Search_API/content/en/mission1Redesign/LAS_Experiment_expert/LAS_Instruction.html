Drawing conclusions from data

In the process of becoming an expert you will encounter many questions. Some of these questions can readily be answered from the existing literature. Other questions need to be examined experimentally, and that's what this activity is all about.

The ultimate goal of this activity is to test your hypotheses and answer your research questions on the basis of self-conducted experiments. As you can see in the picture below, this process comprises four steps:

  • design experiments, using the experimental procedure tool;
  • run experiments, using simulations or real equipement;
  • process data, using the data processing tool;
  • draw conclusions, and write them down in the text editor.

How to start?

You start this activity by designing an experiment for each of your hypotheses. Open the object Hypotheses of my expert group and the Experimental procedure tool to get going.
