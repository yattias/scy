Traiter vos données

Si vous avez recueilli un nombre important de données, c’est souvent une bonne idée de les réorganiser de manière structurée et pratique. Vous pouvez présenter vos données de telle manière qu’il soit facile de comprendre quelles mesures vous avez faites et quel est le résultat de votre expérimentation.

Par exemple, vous pouvez:

- Calculer les valeurs moyennes (et peut-être les écarts-types)
- Présenter les données les plus pertinentes dans des tableaux
- Représenter vos variables dépendantes et indépendantes sous forme de diagramme
- Utiliser un test statistique simple pour évaluer la validité de vos conclusions

Maintenant il temps de revenir à vos questions de recherche et hypothèses pour voir si vous pouvez tirer des conclusions à partir de vos données.
