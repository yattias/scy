Analyse critique et comparée de ressources

Après avoir trouvé un certain nombre de ressources avec des informations appropriées, vous devez rassembler et organiser les informations de façon efficace. Les conseils suivants peuvent vous être utiles :

• Lire.

D’abord, il faut lire avec attention les documents trouvés. Bien sûr, vous pouvez sauter les passages qui n’ont rien à voir avec vos questions de recherche.

• Résumer.

Ensuite, il est nécessaire de résumer les informations dont vous avez besoin. Plutôt que d’écrie un résumé sous forme de texte, vous pouvez aussi essayer de faire une liste de mots clés ou phrases clés. NE PAS copier-coller des paragraphes entiers du texte initial.

• Organiser les informations.

Parfois, il est judicieux d’organiser des informations particulières dans des tableaux (par exemple, liste d’appareils ménagers et leur fonction, etc...).
