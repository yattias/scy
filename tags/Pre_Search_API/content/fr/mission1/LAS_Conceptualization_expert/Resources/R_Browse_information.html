Recherche documentaire

Un rôle spécifique d’expert vous a été attribué : expert en énergie thermique, expert en sources d’énergie et expert en utilisation domestique. Maintenant vous devez faire des lectures générales autour du sujet, pour trouver quels sont les concepts et phénomènes importants dans le domaine d’expert étudié. Peut-être que vous avez déjà beaucoup d’expérience pour ce type de recherche. Néanmoins les indications suivantes peuvent vous être utiles pour récolter les bonnes informations.

• Ne pas se fier exclusivement à une ressource.

Wikipedia est souvent un bon site web pour démarrer des recherches sur un sujet qui est nouveau pour vous. Cependant, il est également nécessaire de chercher des informations via d’autres sources (autres sites web, livres, etc.). L’information trouvée sur un site web particulier est souvent incomplète, et parfois même incorrecte. Une autre source d’information pertinente correspond aux personnes expertes du domaine étudié. Aussi, vous pouvez envisager de faire des interviews pour récupérer des informations spécifiques.

• Evaluer la fiabilité de votre ressource.

Certaines ressources (surtout des sites web) ne fournissent pas des informations fiables. Néanmoins, les sites web d’universités, musées, organisations en lien avec le gouvernement sont en général très fiables. Utiliser votre bon sens pour évaluer qui est l’auteur d’un site web et si vous pouvez faire confiance à l’information fournie.

• Garder toujours à l’esprit la question (de recherche).

La quantité d’information disponible sur internet est presque illimitée. Si vous n’essayez pas de vous focaliser sur votre propre question, vous risquez d’être ‘perdus’ au milieu des informations trouvées. Bien sûr, vous avez toujours la possibilité d’ajouter de nouvelles questions de recherche au fur et à mesure de votre démarche.

• Garder les adresses (URL) de tous les sites web utilisés.

Il arrive souvent de copier des informations d’un site web donné et plus tard de vouloir revenir sur ce site pour copier des informations additionnelles. Par conséquent il est nécessaire de garder une trace de votre ‘historique de recherche’. De plus, une liste complète de toutes les ressources utilisées sera requise dans le rapport (individuel) que vous remettrez à la fin de la mission.
