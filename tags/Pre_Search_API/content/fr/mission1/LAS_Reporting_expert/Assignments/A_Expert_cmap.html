Carte conceptuelle d’expert

Vous travaillez pour devenir un expert dans un domaine particulier. Comme résultat de vos recherches (aussi bien théoriques qu’expérimentales), vous allez acquérir des connaissances sur les principales notions et mécanismes de ce domaine. Dans la deuxième partie de votre mission, vous allez devoir utiliser ces connaissances au bénéfice de votre groupe de conception. Les décisions qui seront pris pendant la phase de conception de la maison dépendront beaucoup de chaque expertise.

L’objectif de cet exercice est de mettre vos conclusions d’expert dans une carte conceptuelle. Une carte conceptuelle est une représentation graphique de notions importantes (dans des boites) et des relations existant entre ces notions (représentées par des flèches sur lesquelles on place un verbe). Le but de faire une carte conceptuelle est double :
- Premièrement, vous ne pouvez pas produire une carte conceptuelle sans avoir vraiment bien compris les notions qu’elle contient. Donc, faire une carte conceptuelle va vous obliger à être précis sur les relations existant entre ces notions.
- Deuxièmement, une carte conceptuelle est un très bon outil pour partager vos conclusions avec d’autres personnes. Vous utiliserez donc la carte conceptuelle pour partager vos connaissances avec votre groupe d’expert puis par la suite avec votre groupe de conception.

Plus précisément, les personnes qui liront votre carte conceptuelle devront être capables de comprendre quels sont les principales notions ainsi que les mécanismes pertinents pour résoudre la mission, et agir sur l’émission de CO2 de votre maison.
