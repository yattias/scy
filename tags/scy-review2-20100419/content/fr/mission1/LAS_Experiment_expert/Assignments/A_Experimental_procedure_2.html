Quelques conseils…

Etant donné que la nature des expérimentations varie entre les différents groupes ou les différentes questions, il n'est pas possible de vous donner des règles très précises pour vous aider dans votre conception expérimentale. Cependant, vous pouvez vous appuyer sur la liste de critères suivante dans l'optique d'évaluer votre protocole expérimental :

Le protocole expérimental est clair et précis.
• La liste des tâches est complète. Tous les matériels et paramètres sont indiqués.
• La liste des tâches est structurée d'une façon qui facilite la compréhension. Les tâches peuvent ainsi être organisées en étapes et sous-étapes.

L'expérimentation est exécutable.
• Le matériel choisi est disponible.
• L'expérimentation peut être exécutée dans le temps imparti.
• Les risques potentiels sont gérés.

L'expérimentation est cohérente avec vos questions de recherche et hypothèses.
• Les variables décrites dans la question de recherche et/ou dans les hypothèses correspondent aux paramètres qui sont modifiés au cours de l'expérimentation.

L'expérimentation est cohérente avec les méthodes scientifiques usuelles.
• Une seule variable est modifiée pour chaque série de mesures.
• Les erreurs de mesure sont minimisées et les facteurs de variation sont contrôlés. La dispersion des différentes valeurs obtenues lors de mesures répétées doit être la plus faible possible.
