SCYDataViewer est un outil pour le traitement de vos données numériques : vous pouvez les organiser, les traiter avec des fonctions mathématiques, les visualiser avec différents types de graphiques (graphes XY, graphiques à barres, diagrammes circulaires ...).

Vos données doivent être organisées en colonnes : celles-ci peuvent soit contenir des noms (pour vos étiquettes de graphique par exemple), soit contenir des nombres. Dans ce dernier cas, n'oubliez pas d'indiquer l'unité des valeurs dans l'en-tête de la colonne.

Voici les principales actions que vous pouvez appliquer à vos données :

Organiser les données en triant les valeurs d'une colonne,
Choisir d'ignorer des données si elles semblent absurbes ou non pertinentes,
  Appliquer des fonctions sur les lignes ou sur les colonnes
  sélectionner le minimum,
  sélectionner le maximum,
  calculer la somme,
  calculer la moyenne.

Cliquer sur pour ajouter un graphique, et choisir le type de graphique que vous voulez créer (graphe XY, graphiques à barres, diagrammes circulaires...).
Pour un graphe XY, vous pouvez spécifier les paramètres en cliquant sur . Par défaut, vous pouvez faire un zoom manuel (cliquer pour sélectionner la zone de zoom , double-cliquer pour dé-zoomer). Si vous cliquer sur le bouton , cela permet un déplacement manuel dans la fenêtre graphique.

Le bouton vous permet d'appliquer un calcul automatique de l'échelle (basée sur les positions des points).

Le bouton vous permet d'ajouter jusqu'à trois courbes avec des paramètres multiples. Par exemple pour tracer une ligne, vous pouvez taper: f(x) = a*x+b. Des boîtes s'ouvrent, vous permettant de choisir les valeurs de vos paramètres (les valeurs de a et b dans cet exemple): . Les flèches sur la droite des boîtes permet d'augmenter ou de diminuer la valeur de la dernière décimale du paramètre.
Si vous souhaitez faire correspondre vos données avec une courbe, vous pouvez ajuster les paramètres de la fonction afin de minimizer la distance entre le modèle et les données. Cette distance est calculée et affichée en-dessous des paramètres avec la couleur de la courbe correspondante.
