Fitex is a tool for processing your numerical data: you can organize them, treat them with mathematical functions, visualize them with different kind of graphs (2D plots, bars, pies...).

You can:

  • Organize the dataset by sorting values of a column,
  • Choose to ignore some data if they seem absurd or non relevant,
  • , , , Apply some functions on the rows or on the columns.

Click on in order to add a graph: you choose the type of the graph you want to create and the data that will be plotted.

For a XYGraph, you can specify the parameters of the graph by clicking on . There are 3 different modes to configure the graph window:

  • Automatic scale (based on the plots positions),
  • Move on the graph,
  • Manual zoom (click to zoom, double click to unzoom).

The button allows you to add up to functions with multiples parameters: thus you can see your experimental data fitted with one or more curves. You can adjust manually the parameters of the functions and the tools calculates the distance between the model and the data.
