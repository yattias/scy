Fitex est un outil pour le traitement de vos données numériques : vous pouvez les organiser, les traiter à l'aide de fonctions mathématiques, les visualiser avec différents types de graphes (nuage de points, graphiques à barres, diagramme circulaire...).

Vous pouvez :

  • Organiser vos données en triant les valeurs d'une colonne,
  • Choisir d'ignorer certaines données si elles vous semblent absurdes ou non pertinentes,
  • , , , Appliquer des fonctions mathématiques sur les lignes ou les colonnes.

Cliquez sur pour ajouter un graphe : vous pouvez alors choisir le type de graphe que vous voulez créer et les données qui seront tracées.

Pour un graphe XY, vous pouvez spécifier les paramètres du graphe en cliquant sur . Il y a 3 modes différents pour configurer la fenêtre du graphe :

  • Echelle automatique (basée sur les positions des points),
  • Déplacement sur le graphe,
  • Zoom manuel (cliquez pour zoomer, double-cliquer pour dé-zoomer)

Le bouton vous permet d'ajouter jusqu'à 3 fonctions avec plusieurs paramètres : ainsi vous pouvez voir vos données expérimentales par rapport à une ou plusieurs courbes. Vous pouvez ajuster manuellement les paramètres des fonctions et l'outil calcule la distance entre le modèle et les données.
