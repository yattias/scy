Manuel d'utilisation de la simulation thermique d'une maison

Avec cette simulation thermique vous pouvez explorer comment construire une maison à basse consommation d'énergie et à faible emission de CO2. Vous pouvez modifier les paramètres de base de la maison et voir comment cela influence le coefficient de déperdition énergétique de la maison. Cette simulation comporte 3 parties :
1) une animation de maison
2) des paramètres d'entrée et de sortie
3) une représentation graphique des données


Figure. Ecran de l'onglet pour les murs (Walls)

La zone de réglage des paramètres est composée de plusieurs thèmes : Général, Murs (Walls), Toit (Roof), Plancher (floor), Fenêtres (Windows), Ventilation, Données de la maison (House data) et Température. Vous pouvez modifier les valeurs de chaque thème et obtenir les résultats correspondant sous forme numérique et graphique.

Table. Paramètres d'entrée et de sortie des thèmes.

In order to facilitate navigation in simulation, all the input parameters are marked as pink ones and the output values have green background color.

Instruction

• Choose input values (wall area, structure material, etc.)
• Look at the results. On the graphical visualization you can see heat loss coefficient for each part of the house. If you change the input values of the house, the changes will be instantly visible on the graph.
• You can change the input values one at a time without resetting all the data.
• If you still want to delete all the inserted data and start again, you can easily reset all the values by clicking on the button “Reset” and enter new data.
• The topic Data gives you full overview of all input and output data of the house

Although the house thermal simulation is simplified and all influencing factors are not taken into account, it still gives you a basic overview of building a thermally efficient house.

NB!

Remember, there are also limitations in input values to prevent simulating unrealistic houses.
