Concevoir un protocole expérimental: quelques infos

Un protocole expérimental est la description précise, pas après pas, d'une expérience. Même si vous avez prévu de réaliser vous même l'expérience, le protocole doit être précis et clair. Une autre personne doit avoir suffisamment de détails pour pouvoir réaliser la même expérience dans les même conditions.

Un protocole expérimental est composé d'une liste de tâches organisées en ordre chronologiques. Tout le matériel nécessaire ainsi que les paramètres qui définissent une tâche doivent être spécifiés. Par exemple, la quantité de chaque substance, les réglages d'un appareil de mesure...).

Exemple de tâche (le matériel et les paramètres sont soulignés):
Prélever 50 mL de solution S avec une pipette de 50 mL.

Référez-vous à la liste suivante pour évaluer votre protocole expérimental:

Le protocole expérimental est précis et clair.

  • La liste des tâches est complète. Tout le matériel et les paramètres sont spécifiés.
  • La liste des tâches est lisible est facilement compréhensible. Les tâches peuvent être organisées en étapes et sous étapes.

    • Le protocole expérimental est réalisable.
  • Le matériel choisi est disponible.
  • L'expérience peut être réalisée dans les temps impartis.
  • Les imprévus ont été pris en compte.

    • L'expérience est cohérente avec votre question de recherche et avec vos hypothèses.
  • Les variables décrites dans votre question de recherche sont les paramètres qui varieront au cours de l'expérience.

    • L'expérience suit la démarche scientifique.
  • Vous ferez varier seulement 1 variable à la fois.
  • Les erreurs de mesure sont minimisées, les sources de variation sont contôlées. La dispersion des différentes valeurs obtenues doit être la plus faible possible.
