Επεξεργασία δεδομένων

Αν έχετε συλλέξει μεγάλο αριθμό αριθμητικών δεδομένων, είναι συχνά μια καλή ιδέα να τα αναδιοργανώσετε με έναν δομημένο και κατάλληλο τρόπο. θέλετε να παρουσιάσετε τα δεδομένα σας με τέτοιο τρόπο, ώστε να είναι εύκολο να καταλάβει κάποιος ποιες μετρήσεις έχετε κάνει και τι αποτελέσματα έδωσαν τα πειράματά σας.

Προσέξτε να έχετε κάνει τα ακόλουθα:

  • υπολογίστε τις μέσες τιμές (και ίσως και τις τυπικές αποκλίσεις)
  • παρουσιάστε τα πιο σχετικά δεδομένα σε μικρούς πίνακες 
  • φτιάξτε διαγράμματα για τις ανεξάρτητες και εξαρτημένες μεταβλητές
  • χρησιμοποιήστε ένα απλό στατιστικό έλεγχο για να αξιολογήσετε την αξιοπιστία των συμπερασμάτων σας

Τώρα ήρθε η στιγμή να πάτε πίσω στα ερευνητικά σας ερωτήματα και τις υποθέσεις σας και να δείτε αν μπορείτε να εξάγεται οποιαδήποτε συμπεράσματα από τα δεδομένα σας.
