Συμπεράσματα από τα πειράματα των ειδικών

Εξαγωγή συμπερασμάτων

Αυτό που προκύπτει από τα πειράματά σας πρέπει τώρα να συγκριθεί με τις υποθέσεις που έχετε διατυπώσει προηγουμένως. Εξετάζοντας τα δεδομένα, προσπαθήστε να εξάγετε τα συμπεράσματα για κάθε ένα από τα ερευνητικά σας ερωτήματα. Κάθε υπόθεση μπορεί να είναι σωστή ή λανθασμένη (ή μερικώς λανθασμένη).

  • Αν η υπόθεσή σας αποδεικνύεται σωστή, μπορείτε να προχωρήσετε σε άλλα ερευνητικά ερωτήματα. Μερικές φορές τα δεδομένα σας σάς δίνουν αφορμή να σκεφτείτε άλλα ερευνητικά ερωτήματα.
  • Αν η υπόθεσή σας αποδεικνύεται λανθασμένη, αυτό δε σημαίνει ότι ολόκληρη η έρευνα σας δεν ήταν επιτυχής. Γνωρίζοντας ότι η υπόθεση σας είναι λάθος, επίσης αποτελεί «νέα γνώση». Μερικές φορές, είναι ακόμη δυνατόν να δοθεί μια απάντηση σε ένα ερευνητικό ερώτημα αμέσως. Σε άλλες περιπτώσεις, είναι απαραίτητα να διατυπωθεί μια νέα υπόθεση και να σχεδιαστούν και να πραγματοποιηθούν νέα πειράματα.

Ένα άλλο συμπέρασμα, μπορεί να είναι ότι τα δεδομένα σας είναι ανεπαρκή για να εξαχθούν οποιαδήποτε οριστικά συμπεράσματα. Τότε, μια καλή ιδέα θα ήταν να αλλάξει ο πειραματικός σχεδιασμός σας ή/και να πραγματοποιηθούν περισσότερα πειράματα.

Εκτός από το να γράψετε τα συμπεράσματά σας σε ένα συνεχές κείμενο, πρέπει επίσης να χρησιμοποιήσετε τη γνώση που αποκτήσατε για να κατασκευάσετε ή να επεκτείνετε τον εννοιολογικό σας χάρτη.
