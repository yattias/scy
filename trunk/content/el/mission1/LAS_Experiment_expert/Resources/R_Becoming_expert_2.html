Πώς να χωρίσετε την εργασία

Ακολουθώντας αυτά τα βήματα, η ομάδα ειδικών σας ενισχύει τη γνώση για ορισμένες πτυχές του θέματος του σπιτιού που είναι φιλικό στο περιβάλλον ως προς τις εκπομπές διοξειδίου του άνθρακα. Αργότερα κάθε ένας από σας θα χρησιμοποιήσει αυτή τη γνώση προς όφελος της ομάδας σχεδιασμού του. Αλλά για την ώρα πρέπει να λειτουργήσετε σαν ομάδα. Θα ήταν προτιμότερο να διαιρέσετε τους στόχους μεταξύ σας. Αλλά μέχρι το τέλος της φάσης που δουλεύετε σαν ειδικοί θα πρέπει να μοιραστείτε μεταξύ σας ότι έχετε βρει, έτσι ώστε κάθε μέλος να έχει αποκτήσει λίγο πολύ την ίδια γνώση και δεξιότητες.


Τι πρέπει να παραδοθεί

Οι επιστήμονες μοιράζονται συχνά τα αποτελέσματα από την έρευνα τους με μια γραπτή έκθεση. Αντί να γράψετε μια γραπτή έκθεση θα παραδώσετε σαν ομάδα ειδικών δύο προϊόντα που θα εξυπηρετούν λίγο ή πολύ τον ίδιο σκοπό με μια γραπτή έκθεση:

  • Πρώτον, θα παρουσιάσετε τα συμπεράσματα σας σε έναν εννοιολογικό χάρτη. Αυτός ο εννοιολογικός χάρτης  πρέπει να εξηγεί τις σχέσεις μεταξύ των μεταβλητών που έχετε διερευνήσει.
  • Δεύτερον, θα κάνετε μια σύντομη παρουσίαση για τα συμπεράσματα σας στην ομάδα σχεδιασμού σας.


Από πού να αρχίσετε

Πρώτα πρέπει να γίνετε ειδικοί. Μια καλή ιδέα θα ήταν να αρχίσετε με την ανάγνωση κάποιων σχετικών πληροφοριών και μελέτη των πηγών που ήδη υπάρχουν και αφορούν το θέμα σας. Μετά που θα έχετε εξοικειωθεί με τους σημαντικότερους όρους και έννοιες για το κομμάτι της γνώσης σας, θα πρέπει να διατυπώσετε μερικά ερευνητικά ερωτήματα. Αν το ερώτημα σας μπορεί να απαντηθεί πειραματικά, η επόμενη ακολουθία βημάτων θα σας καθοδηγήσει για να διεξάγετε τη διαδικασία και την εκτέλεση του πειράματος και να εξάγετε συμπεράσματα από τα πειράματα αυτά. Όπου απαιτείται ζητήστε από τον εκπαιδευτικό να σας δώσει ανατροφοδότηση.
