Αναζήτηση πηγών πληροφοριών

Σας έχει ανατεθεί ένας συγκεκριμένος ρόλος ειδικού: ειδικοί σε θέματα ενέργειας, ειδικοί σε θέματα μόνωσης και ειδικοί σε θέματα ηλεκτρικής ενέργειας. Τώρα θα πρέπει να διαβάσετε κάποιες σχετικές πληροφορίες, για να μάθετε ποιες έννοιες και φαινόμενα είναι σημαντικά στο συγκεκριμένο τομέα γνώσης με τον οποίο θα ασχοληθείτε. Ίσως να έχετε ήδη πολλή εμπειρία κάνοντας αυτού του είδους την έρευνα. Εντούτοις, οι ακόλουθες οδηγίες μπορεί να σας βοηθήσουν να συλλέξετε τις σωστές πληροφορίες:

Μην στηριχθείτε αποκλειστικά σε μια πηγή.

Το Wikipedia είναι συχνά μια καλή ιστοσελίδα για να αρχίσετε με ένα θέμα που είναι νέο σε εσάς. Εντούτοις, είναι απαραίτητο να συλλεχθούν πληροφορίες από διάφορες άλλες πηγές (άλλες ιστοσελίδες, βιβλία, κ.λπ.). Οι πληροφορίες που βρίσκετε σε μια ιστοσελίδα είναι συχνά ελλιπής, και ακόμη μπορεί να είναι λανθασμένες. Μια άλλη καλή πηγή πληροφοριών είναι μια ομάδα ανθρώπων που είναι ειδικοί. Έτσι θα μπορούσατε επίσης να θεωρήσετε μια συνέντευξη με κάποιον ειδικό ως μια πηγή συγκεκριμένων πληροφοριών.

Αξιολογήστε την αξιοπιστία της πηγής σας.

Κάποιες πηγές (ειδικά ιστοσελίδες) δεν παρέχουν ακριβείς πληροφορίες. Ιστοσελίδες από πανεπιστήμια, μουσεία και οργανισμούς που έχουν σχέση με την κυβέρνηση συνήθως είναι αξιόπιστες πηγές πληροφοριών. Χρησιμοποιήστε τη κοινή σας λογική για να αξιολογήσετε ποιος είναι ο συντάκτης μιας ιστοσελίδας, και κατά πόσο οι πληροφορίες στην συγκεκριμένη ιστοσελίδα μπορούν να είναι άξιες εμπιστοσύνης.

Πάντα να έχετε στο μυαλό σας τα (ερευνητικά σας) ερωτήματα.

Η ποσότητα των διαθέσιμων πληροφοριών μέσω του διαδικτύου είναι σχεδόν απεριόριστη. Εάν δεν επικεντρωθείτε στα συγκεκριμένα ερωτήματα που έχετε, θα είναι εύκολο να χαθείτε σε όλα τα δεδομένα και τις εικόνες που θα βρείτε στο διαδίκτυο. Φυσικά μπορείτε να προσθέσετε νέα ερευνητικά ερωτήματα κατά τη διάρκεια αναζήτησης.

Να αποθηκεύετε όλες τις διευθύνσεις (URL’s) των ιστοσελίδων που χρησιμοποιείτε.

Μπορεί να τύχει συχνά να αντιγράφετε κάποιες πληροφορίες από κάποια συγκεκριμένη ιστοσελίδα και – σε κάποιο μετέπειτα στάδιο- μπορεί να αποφασίσετε ότι χρειάζεστε κι άλλες πληροφορίες από τη συγκεκριμένη ιστοσελίδα. Επομένως είναι απαραίτητο να παρακολουθείτε την πορεία του ιστορικού επισκέψεων ιστοσελίδων (browsing history) σας. Επίσης, μια πλήρη λίστα των πηγών που θα έχετε χρησιμοποιήσει θα πρέπει να αποτελεί κομμάτι της (ατομικής) αναφοράς που θα πρέπει να παραδώσετε στο τέλος της αποστολής.
