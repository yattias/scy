Σχεδιασμός πειραματικής διαδικασίας

Πριν πραγματοποιήσετε παιχνίδι μοντελοποίησης, πρέπει να περιγράψετε τις διάφορες ενέργειες που πρέπει να εκτελέσετε προκειμένου να τρέξετε το πείραμά σας. Η διαδοχή των βημάτων που θα ακολουθήσετε ονομάζεται πειραματική διαδικασία. Η περιγραφή της πειραματικής διαδικασίας πρέπει να είναι λεπτομερής. Ένα άλλο άτομο θα πρέπει να έχει επαρκή καθοδήγηση για να τρέξει το ίδιο πείραμα ακολουθώντας την ίδια πειραματική διαδικασία. Θα χρησιμοποιήσετε το εργαλείο πειραματικού σχεδιασμού στο SCY-Lab. Το εργαλείο ζητά να γράψετε τα διερευνήσιμα ερωτήματα και τις υποθέσεις, που έχετε ήδη διατυπώσει. Έπειτα θα περιγράψετε τη γενική αρχή του πειράματος σας, όπου θα χρειαστεί να καθορίσετε τις μεταβλητές που θα εξετάσετε κατά τη διάρκεια του πειράματος. Σημειώστε ότι πρέπει να καθορίσετε την εξαρτημένη μεταβλητή (δηλαδή την παράμετρο η οποία αντιστοιχεί στο αποτέλεσμα που θα παρατηρήσετε στο πείραμα) και μία ή περισσότερες ανεξάρτητες μεταβλητές (δηλαδή τις παραμέτρους που θα διαχειριστείτε προκειμένου να παρατηρήσετε την επίδραση τους στη εξαρτημένη μεταβλητή). Για παράδειγμα, μπορείτε να διερευνήσετε την επίδραση διαφορετικών ποσοτήτων νερού που θα παρέχετε σε ένα φυτό (ανεξάρτητη μεταβλητή: νερό ποτίσματος που μετριέται σε λίτρα) στην ανάπτυξη του φυτού που μπορεί να παρατηρηθεί από την αλλαγή στο ύψος του φυτού (εξαρτημένη μεταβλητή: ύψος του φυτού που μετριέται σε εκατοστόμετρα).

Τώρα πρέπει οργανώσετε την πειραματική σας διαδικασία.

Αυτό είναι ένα ειδικό εργαλείο το οποίο θα σας βοηθήσει με αυτή τη διαδικασία. Πρώτα ανοίξτε τις υποθέσεις σας και αντιγράψτε το περιεχόμενο τους για να το επικολλήσετε στο πεδίο των υποθέσεων της πειραματικής σας διαδικασίας. Μπορείτε επίσης να διαβάσετε κάποιες οδηγίες για το σχεδιασμό πειραμάτων.
