Οδηγίες: Λύση προβλήματος

1. Αν η υπόθεσή σας αποδειχθεί σωστή, μπορείτε να προχωρήσετε σε άλλο διερευνήσιμο ερώτημα. Κάποιες φορές τα δεδομένα σας οδηγούν σε νέα διερευνήσιμα ερωτήματα.
2. Αν η υπόθεσή σας αποδειχθεί λανθασμένη, αυτό δε σημαίνει ότι όλη η έρευνα δεν ήταν επιτυχής. Το να είναι η υπόθεσή σας λανθασμένη, αποτελεί επίσης «νέα γνώση». Μερικές φορές είναι πιθανό να απαντήσετε το διερευνήσιμο σας ερώτημα απευθείας. Σε άλλες περιπτώσεις, είναι αναγκαίο να κάνετε νέα υπόθεση, και να σχεδιάσετε και να εκτελέσετε νέα πειράματα. 
3. Άλλο συμπέρασμα μπορεί να είναι ότι τα δεδομένα που έχετε να είναι ανεπαρκή για να σας οδηγήσουν σε σίγουρα συμπεράσματα. Τότε θα ήταν καλή ιδέα να αλλάξετε το σχεδιασμό του πειράματος και/ή να εκτελέσετε νέα πειράματα.
