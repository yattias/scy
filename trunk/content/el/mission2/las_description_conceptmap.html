Εδώ μπορείτε να συνθέσετε τον εννοιολογικό σας χάρτη για τα οικοσυστήματα του γλυκού νερού. Ακολουθήστε τις οδηγίες που δίνονται στο συρτάρι. Όταν συμπληρώσετε τον αρχικό εννοιολογικό σας χάρτη, ξεκινήστε με τον πρώτο κύκλο διερεύνησης.

Θα επανέλθετε σε αυτό το χώρο όταν τελειώσετε με τον πρώτο κύκλο διερεύνησης, για να συμπληρώσετε τον εννοιολογικό σας χάρτη με τις καινούριες γνώσεις που αποκτήσατε κατά τη διερεύνησή σας. Ο εννοιολογικός χάρτης μπορεί να αναπτύσσεται μετά την ολοκλήρωση κάθε κύκλου διερεύνησης.

 

Μαθησιακοί στόχοι

Μετά την επιτυχή ολοκλήρωση των εργασιών σας θα είστε σε θέση να:

• περιγράφετε συνοπτικά τα κύρια βιοτικά και αβιοτικά συστατικά του γλυκού νερού
• περιγράφετε τις σχέσεις μεταξύ των κύριων συστατικών ενός οικοσυστήματος του γλυκού νερού
• προσδιορίζετε και να περιγράφετε τους κύριους παράγοντες που επιδρούν σε ένα οικοσύστημα του γλυκού νερού.
