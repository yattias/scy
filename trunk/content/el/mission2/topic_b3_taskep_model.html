Σχεδιασμός πειραματικής διαδικασίας (μοντέλο SCYDynamics)

Πριν πραγματοποιήσετε το παιχνίδι προσομοίωσης, πρέπει να περιγράψετε τις διάφορες ενέργειες που πρέπει να εκτελέσετε προκειμένου να τρέξετε το πείραμά σας. Η διαδοχή των βημάτων που θα ακολουθήσετε ονομάζεται πειραματική διαδικασία. Η περιγραφή της πειραματικής διαδικασίας πρέπει να είναι λεπτομερής. Ένα άλλο άτομο θα πρέπει να έχει επαρκή καθοδήγηση για να τρέξει το ίδιο πείραμα ακολουθώντας την ίδια πειραματική διαδικασία. Θα χρησιμοποιήσετε το εργαλείο πειραματικού σχεδιασμού στο SCY-Lab. Το εργαλείο ζητά να γράψετε τα ερευνητικά ερωτήματα και τις υποθέσεις σας, που έχετε ήδη διατυπώσει. Έπειτα θα περιγράψετε την γενική αρχή του πειράματος σας, όπου θα χρειαστεί να καθορίσετε τις μεταβλητές που θα εξετάσετε κατά τη διάρκεια του πειράματος. Σημειώστε ότι πρέπει να καθορίσετε την εξαρτημένη μεταβλητή (δηλαδή την παράμετρο η οποία αντιστοιχεί στο αποτέλεσμα που θα παρατηρήσετε στο πείραμα) και μία ή περισσότερες ανεξάρτητες μεταβλητές (δηλαδή τις παραμέτρους που θα διαχειριστείτε προκειμένου να παρατηρήσετε την επίδραση τους στη εξαρτημένη μεταβλητή). Για παράδειγμα, μπορείτε να διερευνήσετε την επίδραση διαφορετικών ποσοτήτων νερού που θα παρέχετε σε ένα φυτό (ανεξάρτητη μεταβλητή: νερό ποτίσματος που μετριέται σε λίτρα) στην ανάπτυξη του φυτού που μπορεί να παρατηρηθεί από την αλλαγή στο ύψος του φυτού (εξαρτημένη μεταβλητή: ύψος του φυτού που μετριέται σε εκατοστόμετρα).

Οργανώστε τις ενέργειας σας σε διαδοχικά βήματα. Κάθε βήμα θα περιλαμβάνει ένα αριθμό ενεργειών. Σημειώστε ότι αν έχετε περισσότερες από μια ανεξάρτητη μεταβλητή, θα πρέπει να αλλάζετε μόνο μία από αυτές κάθε φορά, έτσι ώστε αυτό που θα παρατηρήσετε να συμβαίνει στην εξαρτημένη μεταβλητή να οφείλεται σε επίδραση μόνο εκείνης της ανεξάρτητης μεταβλητής. Καθώς καθορίζετε τα ερωτήματα, τις υποθέσεις, τις αρχές, τα βήματα και τις ενέργειες μπορείτε να προσθέσετε σχόλια τα οποία πιστεύετε ότι θα κάνουν την πειραματική διαδικασία περισσότερο κατανοητή.

Τώρα πρέπει οργανώσετε την πειραματική σας διαδικασία.         

Αυτό είναι ένα ειδικό εργαλείο το οποίο θα σας βοηθήσει με αυτή τη διαδικασία. Πρώτα ανοίξτε τις υποθέσεις σας και αντιγράψτε το περιεχόμενο τους για να το επικολλήσετε στο πεδίο των υποθέσεων της πειραματικής σας διαδικασίας. Αν τα ερευνητικά ερωτήματα και οι υποθέσεις που διατυπώσατε προηγουμένως δεν μπορούν να απαντηθούν με τα εργαλεία και τον εξοπλισμό που είναι διαθέσιμος, θα πρέπει να τα διορθώσετε και να γράψετε αναθεωρημένα ερωτήματα και υποθέσεις στην εργασία σας για την πειραματική διαδικασία. Σε αυτό το σημείο μπορείτε να αναζητήσετε τα μαθησιακά προϊόντα των συμμαθητών σας με τους οποίους συνεργάζεστε προκειμένου να ολοκληρώσετε τη δική σας πειραματική διαδικασία. Μπορείτε επίσης να διαβάσετε μερικές οδηγίες για το σχεδιασμό πειραμάτων.
