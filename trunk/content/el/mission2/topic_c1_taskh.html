Διατύπωση υποθέσεων

Έχετε ήδη διατυπώσει ερευνητικά ερωτήματα. Γράφοντας τα ερευνητικά ερωτήματα πιθανόν να έχετε κάποιες ιδέες για το ποιες μπορεί να είναι οι απαντήσεις. Αυτές οι ιδέες είναι οι υποθέσεις σας. Επόμενος σας στόχος είναι να διατυπώσετε μια υπόθεση για κάθε ένα από τα ερωτήματα. Η υπόθεση είναι μια πρόταση η οποία έχει συνήθως τη δομή «αν…τότε…».

Αν σας ενδιαφέρει να μάθετε πώς γράφεται μια «καλή» υπόθεση τότε μπορείτε να διαβάσετε μερικές οδηγίες.
