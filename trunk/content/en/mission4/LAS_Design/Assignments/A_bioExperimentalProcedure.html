Experimental procedure

As you know you have six DNA samples that need to be tested: one DNA sample from the crime scene and five reference samples to investigate if there is a match with the DNA from the crime scene.

You have to take the following steps:
1. Drag-and-drop the icon of the "Goal of the analysis" you have written before to the "Research question" field.
2. Write your hypothesis by finishing the sentence.
3. Describe the principle of the manipulation by using the previous step "Choice of the techniques".
4. Read the description of the material you are going to use during the experiment.
5. Read the "Manipulation and data treatment" section step by step. You will see some questions in blue. Don't forget to answer them in the same location.
