Import your data that was collected with the Vernier device. Put the memory card in particular slot of the computer and click on the icon ‘Add data to the current datasheet…’ and search for the file containing the data (usually *.gmbl).

After the data have been imported delete unnecessary columns and analyse the data. You can also create a graph. Click on the icon ‘Create a new graph’.

If you cannot import your data then you can use the pre-defined columns of the table and fill in them manually.
