It’s time to start experimenting. Open your ‘Hypotheses ELO’ and collect evidence with regard to your hypotheses using this prey-predator model.

  1. First, look at the model.
  2. Next, try to draw graphs according to your hypotheses. To do this, click on the tab 'Graph', select appropriate variables and click 'run simulation'.
  3. Finally, produce a data table by using your experimental procedure to select variables and run simulations.

Note that you should manipulate only the independent variables that you have identified in the description of your experimental procedure.
