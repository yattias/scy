== DESCRIPTION ==

GStyler is a GWT based Graph-Modelling tool.
It aims to be the Web-Version of Freestyler (http://www.collide.info/index.php/FreeStyler)

Contact: roth@collide.info

== LEGAL == 

* Drawing the edges is realised using the gwt-diagrams library (http://code.google.com/p/gwt-diagram/)
* Drag&Drop support is done usind gwt-dnd (http://code.google.com/p/gwt-dnd/)
