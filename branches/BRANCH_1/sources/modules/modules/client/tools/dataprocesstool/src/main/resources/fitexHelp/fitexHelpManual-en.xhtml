SCYDataViewer is a tool for processing your numerical data: you can organize them, treat them with mathematical functions, visualize them with different kind of graphs (XY graph, bars, pies...).

Here are the main actions you can apply to you data:

Organize the dataset by sorting values of a column,
Choose to ignore some data if they seem absurd or non relevant
  Apply some functions on the rows or on the columns
  select minimum
  select maximum
  calculate the sum
  calculate the average

Click on in order to add a graph, and choose the type of the graph you want to create (XY graph, bars, pies...).
For a XYGraph, you can specify its parameters by clicking on . There are 3 different modes to configure the graph window:

automatic scale (based on the plots positions)
manual move of the graph window
manual zoom (click to select the zooming zone, double click to unzoom)

The button allows you to add up to three curves with multiple-parameters functions. For example, to draw a line, you can type : f(x) = a*x+b. Boxes will open, allowing you to choose the values of your parameters (the values of a and b for the latter example): . The arrows on the right of the boxes increase or decrease the value of the last decimal place of the parameter.
If you want to fit your data with a curve, you can adjust the parameters of the function in order to minimize the distance between the model and the data. This distance is calculated and is displayed below the parameters, with the color of the curve.
